# TaskPilot

TaskPilot é um website de gerenciamento de tarefas projetado para ajudar você a organizar e acompanhar suas atividades diárias de forma eficiente.

## Funcionalidades

- **Criação de Tarefas:** Adicione novas tarefas com descrições detalhadas.
- **Edição de Tarefas:** Edite tarefas existentes para atualizar informações.
- **Exclusão de Tarefas:** Remova tarefas que não são mais necessárias.
- **Visualização de Tarefas:** Veja todas as suas tarefas em uma interface amigável.
- **Filtragem e Pesquisa:** Encontre tarefas rapidamente usando filtros e pesquisa.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
