# TaskPilot

TaskPilot é um website de gerenciamento de tarefas projetado para ajudar você a organizar e acompanhar suas atividades diárias de forma eficiente.

## Funcionalidades

### TAREFAS

- Criação
- Edição
- Exclusão
- Visualização
- Filtragem e pesquisa

## Tecnologias Utilizadas

- React
- Node
- Webpack, babel
- TypeScript / JSX
- Express
- Zod
- Prisma
